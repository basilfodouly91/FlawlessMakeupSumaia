================================================================================
                    HOW TO VIEW UPLOADED IMAGES
================================================================================

Method 1: Visual Studio
  1. Open Visual Studio
  2. View → SQL Server Object Explorer
  3. Expand: (localdb)\MSSQLLocalDB
  4. Expand: Databases → FlawlessMakeupDB → Tables → Products
  5. Right-click Products → View Data
  6. Scroll to ImageUrl column
  7. You'll see: "data:image/jpeg;base64,/9j/4AAQSk..."

Method 2: SQL Query
  USE FlawlessMakeupDB;
  SELECT Id, Name, 
         LEFT(ImageUrl, 50) AS ImagePreview,
         LEN(ImageUrl) AS ImageSizeBytes
  FROM Products
  WHERE ImageUrl LIKE 'data:image%';

Method 3: API
  http://localhost:5001/api/products
  Look for imageUrl field

Method 4: Website
  http://localhost:4200
  Images display automatically (browser converts Base64 to image)
